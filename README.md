# 🏥 AI Healthcare Chatbot

An AI-powered Healthcare Chatbot developed using Python that provides preliminary health assistance by analyzing user symptoms and generating a downloadable health report.

---

## 📌 Features

- 🤖 Interactive chatbot for symptom-based guidance
- 🩺 Predicts possible diseases from symptoms
- 📋 Generates a detailed PDF Health Report
- 💬 User-friendly conversational interface
- 📂 JSON-based disease dataset
- ⚡ Fast and lightweight Python application

---

## 🛠️ Tech Stack

- Python
- JSON
- ReportLab (PDF Generation)
- VS Code

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Healthcare-Chatbot.git
```

### Navigate to Project

```bash
cd Healthcare-Chatbot
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

---

## 📖 How It Works

1. User enters health symptoms.
2. Chatbot analyzes symptoms.
3. Relevant disease information is retrieved.
4. Health suggestions are displayed.
5. A PDF Health Report is generated.

---

## 📄 Sample Health Report

The application automatically creates a PDF report containing:

- User Information
- Entered Symptoms
- Predicted Disease
- Recommended Precautions
- Date & Time
