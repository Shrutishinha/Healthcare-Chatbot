## 📖 Overview

The **AI Healthcare Chatbot** is a Python-based healthcare assistance application that provides users with preliminary health guidance by analyzing symptoms entered during a conversational interaction.

The chatbot identifies possible health conditions using a structured disease dataset and generates a professionally formatted PDF health report containing predictions, precautions, and recommendations.

This project demonstrates the practical application of Python programming, structured data processing, and automated document generation in the healthcare domain.

---

## ✨ Key Features

- Intelligent symptom-based health assistance
- Interactive conversational chatbot
- Disease prediction using a structured JSON dataset
- Automated PDF health report generation
- Clean and modular Python architecture
- Lightweight and easy to deploy
- Easily extendable disease database
- User-friendly interface

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Shrutisinha/Healthcare-Chatbot.git
```

Navigate into the project directory

```bash
cd Healthcare-Chatbot
```

Install the required dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
